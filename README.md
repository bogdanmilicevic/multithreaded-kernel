# multithreaded-kernel
Built a multithreaded operating system kernel from scratch for the RISC-V architecture. Implemented core
OS features including memory management, thread scheduling, and semaphores. Used RISC-V Assembly to
handle CPU context switching and hardware interrupts (timer and console). Created C and C++ APIs to allow
user programs to communicate with the kernel.
For more details about the project, see the project PDF.
